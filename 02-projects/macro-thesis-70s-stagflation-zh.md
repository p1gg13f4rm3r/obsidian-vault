# 宏观主题：1970年代滞胀 + 中国崛起为超级大国

**日期：** 2026-03-27  
**状态：** 活跃主题

---

## 主题概要

- 美元信誉/储备货币地位加速流失
- 中国已成为超级大国竞争对手
- 滞胀环境（低增长 + 高通胀）
- 伊朗冲突加剧能源价格压力
- 财政赤字飙升，债务/GDP 约 130%
- **美联储无法复制沃尔克模式 — 金融压制是唯一可行之路**

---

## 历史类比：1970年代

### 相同之处
- 石油危机
- 财政赤字
- 美元信誉流失
- 冷战竞争（当时苏联 → 现在中国）

### 不同之处
- 美国能源独立（现在是石油净出口国）
- 债务/GDP 130% vs 1970年代的 35%
- 美元的替代品：人民币、黄金
- 人口结构逆风
- **万物金融化** — 使真正的紧缩几乎不可能

---

## 沃尔克之问：美联储这次能复制吗？

**简短回答：几乎不可能。**

### 1980年为何不同

沃尔克将利率加息至 **20%+** 并保持。那次成功了。但当时：

| 指标 | 1980年 | 今天 |
|------|--------|------|
| 美国国债/GDP | ~35% | ~130% |
| 美国总债务（所有部门） | ~$4万亿 | ~$100万亿+ |
| 标普500市值 | ~$1.3万亿 | ~$50万亿 |
| 抵押贷款市场 | ~$2万亿 | ~$14万亿 |
| 企业债（BBB级） | 很小 | ~$10万亿 |
| 私募股权/杠杆收购 | 几乎没有 | ~$4万亿 |

**一切都已金融化。美联储影响着每一个资产类别。**

### 财政反馈循环

```
利率上升
    → $36万亿+国债利息支出增加
    → 赤字进一步恶化
    → 财政部加大发债
    → 美联储不行动市场收益率也上升
    → 恶性循环
```

目前每年利息支出约 $1万亿+。若现有债务平均利率 5%：**每年 $1.8万亿** — 已超过国防预算。

**美联储不可能加息至 10%+ 而不导致政府破产。**

### 政治经济因素

沃尔克有政治支持。今天的环境：

- **50%+ 美国人持有股票**（401k、ETF）
- **7000万+ 婴儿潮一代** 临近或已退休 — 靠固定收入生活
- **房地产** — 购房者是选民，对利率极度敏感
- **国会** — 不会容忍资产价格下跌 40%

### 美联储实际能做什么

| 情景 | 结果 |
|------|------|
| **完全沃尔克（利率至 15%+）** | 系统崩溃，大萧条 |
| **激进加息（7–8%）** | 衰退，标普 -40–50%，政治上不可行 |
| **渐进加息（4–5%）** | 缓慢、痛苦，需 3–5 年，尚可承受 |
| **金融压制** | 压低实际利率，逐步通胀稀释债务 — **美联储实际做法** |

### 真正的答案

美联储将走 **金融压制** 路线，而非沃尔克：

- 维持利率"足够高"以维护公信力（约 4–5%）
- 允许通胀略微偏高
- 保持实际利率在零附近或轻微负值
- 在 10–20 年内逐步通胀稀释债务

这在政治上可以承受。这也是**隐形税** — 储户和债券持有人无声地补贴政府。

### 对主题的影响

**美联储无法真正沃尔克，反而是黄金的利好：**
- 实际利率受压
- 美元疲软（缓慢走弱，非崩盘）
- 黄金获得多年支撑
- 能源维持高位 — 不会因加息而抑制需求
- 防务股因地缘政治溢价持续走强

**美联储会打压黄金** — 每次黄金涨势过快，他们会暗示加息。黄金是对美联储政策的信任投票。
